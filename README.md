# Linux Scripts 🐧

A collection of bash scripts for system monitoring and automation.

## Scripts

### system_report.sh
Displays a coloured system report including:
- Current user and date
- Disk space usage
- RAM usage
- IP address
- Disk health check

## Usage
chmod +x system_report.sh
./system_report.sh
